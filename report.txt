# Spotify Track Genre Classification
**Competition**: DataSprint — Decode the Beat

## 1. Approach
Advanced Multi-Class Classification Workflow using Class-Balanced Voting Ensemble.
This tackles overlapping clusters using non-linear bounding structures generated by LightGBM and ExtraTrees.

## 2. Preprocessing & Feature Engineering
* **Frequency Encoding:** Enabled purely targeting the distinct 'artists' dataset column prior to standard scaling.
* **Feature Engineering:** Engineered acoustic intersections matching dynamics (e.g., `energy_x_danceability`, `mood_divergence`).
* **Handling Missing Data:** Handled missing categorical/numerical values independently then standardized to maintain equal feature weighting across algorithms.
